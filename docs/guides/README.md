# 使用指南

本文档提供了如何使用和开发这个monorepo项目的指南。

## 开发指南

### 安装依赖

```bash
pnpm install
```

### 开发命令

开发Vue库：
```bash
pnpm dev:vue
```

开发React库：
```bash
pnpm dev:react
```

### 构建命令

构建所有项目：
```bash
pnpm build
```

## 贡献指南

1. Fork 这个仓库
2. 创建你的特性分支 (`git checkout -b feature/amazing-feature`)
3. 提交你的更改 (`git commit -m 'Add some amazing feature'`)
4. 推送到分支 (`git push origin feature/amazing-feature`)
5. 开启一个 Pull Request
